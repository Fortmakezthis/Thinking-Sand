# Thinking-Sand
Repo to store files for a "mod" made with KubeJS

This is made for 1.20.1 but may work with 1.21.1 idk haven't tested it.

This needs Create and Create Optical. Also KubeJS, obviously.

Installation:

Just download the repo and place KubeJS in the root of the Minecraft instance, if it asks to replace files, replace them.
